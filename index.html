Recent work

MLX42 MLX42 is a simple, small and fast graphical windowing API that lets students in the 42 Network Create graphical projects with ease. It started off as an attempt to learn OpenGL with a mission of replacing the old miniLibX library that was in place before. It features rich documentation, ease of use and hight stability.
Codam Coding College As of October 2021 I am part of Codam, a school in the 42 Network. I really wanted to avoid any University after secondary school, and Codam proved to be the right choice!

42, The meaning of life!
Volumetric Galaxy A procedural galaxy which uses volumetric fog particles for nebulae as well as being able to generate star systems with planets that you can visit.

Allows the user to create their very own world and play god!
Console Manager A plugin that lets you add custom console commands and handle their execution via blueprints, no C++ is required!.

Making debugging really easy as well as opening up the console to more developers.
UMG Colorwheel Make your life colorfull! A plugin for Unreal Engine 4 that lets you use a colorwheel and pick a color (duh!)

This was the first plugin/product I released for Unreal Engine.
Basic Utils Library My utils library plugin was completely written in C++ and encompassed everyday functions that I use all the time.

It included sorting algorithms, data serialization, File IO operations & direct array converters.
Project Life RPG Project Life RPG was a freelance job that I took which involved building a mulitplayer role playing game who's origins come from the GTA V role playing mod.

My tasks included working on driveable vehicles, including customization, and a Construction Kit which allowed the user to place furniture in their apartments.
Flax Engine In late december I found out about the Flax Engine and decided that it was a good starting point to go a step beyond simple game development.

As a result I have contributed lots of PRs towards the Flax Engine both source and documentation wise.
